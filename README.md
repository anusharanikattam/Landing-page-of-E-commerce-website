# 🛍️ ShopSmart - Professional E-Commerce Website

## 📌 Project Overview

ShopSmart is a modern and responsive E-Commerce Website developed using HTML, CSS, and JavaScript. The project provides a complete online shopping experience with dynamic product listings, shopping cart functionality, wishlist management, dark mode support, product filtering, and responsive design.

This project demonstrates frontend web development skills and can be used as a , internship project, & learning project.
